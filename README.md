# Portfolio Project.

This project was built with React, React-Router, Express, Node, Webpack, Babel, Helmet and GSAP.

## Installation:

To install run the following code.

```bash
npm install
```

```bash
npm run start:dev
```
This will run the project in the localhost 3000 with SSR.
